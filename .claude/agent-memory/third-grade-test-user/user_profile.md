Jordan is an 8-year-old 3rd grader used as the UX testing persona for the Paadam kids worksheet app.

- Reading level: 3rd grade, struggles with words over 3 syllables or abstract/technical terms
- Tech comfort: Can tap, swipe, and type slowly; familiar with tablets and phones but not tech-savvy
- Attention span: Short — loses interest if something takes more than a few seconds to figure out
- Motivation: Responds well to game-like rewards, bright colors, surprises, and animal characters
- Character preference: Excited by Max (dog) and Luna (cat) characters; Ada (owl) messages are slightly too formal
- Known frustrations: Multi-step configuration screens, adult vocabulary in labels, typing answers on mobile keyboard
