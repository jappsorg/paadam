## 📋 Overview

This document contains detailed specifications for all image assets needed for the Paadam adaptive learning app. Images should be **child-friendly, educational, warm, and inviting** with a consistent art style throughout.

### **Overall Art Direction**

- **Style:** Modern, clean, friendly illustration style (similar to Duolingo/Khan Academy Kids)
- **Color Palette:** Warm, vibrant but not oversaturated
  - Primary: Soft blues (#5B9BD5), warm oranges (#F4A460), gentle greens (#90C695)
  - Secondary: Purple (#9B88C0), yellow (#FFD966), pink (#F4A6B8)
  - Neutrals: Warm whites (#FFF8F0), soft grays (#F0F0F0)
- **Target Age:** 5-12 years old
- **Mood:** Encouraging, playful, warm, intelligent
- **Avoid:** Overly cute/babyish, dark/scary elements, complex details that distract

## 💡 NOTES FOR DESIGNERS

1. **Consistency is Key:** All characters should feel like they belong to the same world/app
2. **Accessibility:** Ensure sufficient color contrast, avoid relying solely on color to convey information
3. **Cultural Sensitivity:** Characters and scenarios should be inclusive and culturally neutral
4. **Scalability:** Designs should look good at various sizes (from 40px to 1024px)
5. **Memory Optimization:** While maintaining quality, optimize file sizes for mobile devices
6. **Future Flexibility:** Design with customization in mind (swappable colors, accessories)
7. **Age Appropriateness:** Keep designs suitable for ages 5-12, not too babyish or too mature

## 📦 SMART ASSET MANAGEMENT STRATEGY

### Problem

Bundling 600+ high-quality images would create a massive app (100-200MB+), which is inefficient when:

- Different grade levels need different visual aids
- Not all students use all characters
- Higher grades don't need basic counting images
- Users have limited device storage

### Solution: Tiered Asset System with On-Demand Downloads

---

### **Tier 1: Core Bundle (Always Included) - ~15-20MB**

**Essential assets that ship with the app:**

#### Characters (Minimal Set)

- **Ada the Owl only** - Base + 6 emotions (7 images @ 1024x1024)
- Character selection thumbnails for all 7 (7 images @ 256x256)
- No animations, no accessories, no evolution stages in core bundle

#### App Infrastructure

- App icon (all required sizes)
- Splash screen
- Tab bar icons (5 icons, 3 states each)
- Basic UI elements (buttons, progress bars, borders)

#### Essential Badges (10 only)

- First Question
- 3-Day Streak
- Addition Master (Bronze)
- Level Up (1, 5, 10)
- Perfect Score
- Quick Start
- Curious Mind
- Persistent Learner

**Total Core Bundle:** ~15-20MB

---

### **Tier 2: Character Packages (Downloaded on First Selection) - ~8-12MB each**

When student selects their character companion, download that character's complete package:

**Package Contents per Character:**

- Base + 6 emotions (if not Ada)
- 6 animation sprite sheets
- 3 evolution stages
- 7-10 accessories
- Character-specific backgrounds

**Download Trigger:** Character selection screen
**Caching:** Permanent (unless user switches characters)
**Fallback:** Show selection screen with loading indicator

**Example:**

```typescript
// When user selects Max the Fox
await downloadCharacterPackage("max-fox", {
  priority: "high",
  progressCallback: (percent) => showProgress(percent),
});
```

---

### **Tier 3: Grade-Level Asset Packs (Auto-downloaded based on grade) - ~5-10MB each**

**K-1st Grade Pack:** "Early Learner Visuals"

- Counting objects (1-20): apples, stars, hearts, animals
- Basic shapes (2D only)
- Number lines (0-10, 0-20)
- Simple word problem scenes (playground, classroom)
- **Size:** ~8MB

**2nd-3rd Grade Pack:** "Elementary Visuals"

- Counting objects (1-100)
- Base-10 blocks
- Fraction circles (halves, thirds, fourths)
- Number lines (0-100)
- Word problem scenes (shopping, birthday, garden)
- **Size:** ~10MB

**4th-5th Grade Pack:** "Intermediate Visuals"

- Fraction visuals (all divisions)
- Fraction bars and number lines
- 3D geometry shapes
- More complex word problem scenarios
- **Size:** ~8MB

**6th+ Grade Pack:** "Advanced Visuals"

- Minimal visual aids (older students need fewer pictures)
- Geometric diagrams
- Charts/graphs for data
- Abstract representations
- **Size:** ~5MB

**Download Trigger:** After character selection, based on student profile grade
**Caching:** Permanent, auto-upgrade when student advances grade level

---

### **Tier 4: Skill-Specific Micro-Packs (Downloaded just-in-time) - ~500KB-2MB each**

For specialized visuals needed only when working on specific skills:

**Micro-Packs:**

### Download Manager Service

```typescript
class AssetDownloadManager {
  /**
   * Download character package when selected
   */
  async downloadCharacterPack(characterId: string): Promise<void> {
    const packageUrl = `${CDN_URL}/character-packs/${characterId}.bundle`;

    // Check if already downloaded
    if (await this.hasPackage(`character-${characterId}`)) {
      return;
    }

    // Download with progress
    await this.downloadPackage({
      id: `character-${characterId}`,
      url: packageUrl,
      priority: "high",
      onProgress: (percent) => this.notifyProgress(percent),
      required: true, // Block until downloaded
    });
  }

  /**
   * Download grade-level visual pack
   */
  async downloadGradePack(grade: Grade): Promise<void> {
    const packName = this.getGradePackName(grade);
    const packageUrl = `${CDN_URL}/grade-packs/${packName}.bundle`;

    if (await this.hasPackage(`grade-${grade}`)) {
      return;
    }

    // Download in background
    await this.downloadPackage({
      id: `grade-${grade}`,
      url: packageUrl,
      priority: "medium",
      background: true, // Don't block user
    });
  }

  /**
   * Download skill micro-pack on-demand
   */
  async downloadSkillPack(skillId: string): Promise<void> {
    const packName = this.getSkillPackName(skillId);
    const packageUrl = `${CDN_URL}/skill-packs/${packName}.bundle`;

    if (await this.hasPackage(`skill-${skillId}`)) {
      return;
    }

    // Download in background while showing fallback content
    this.downloadPackage({
      id: `skill-${skillId}`,
      url: packageUrl,
      priority: "low",
      background: true,
      optional: true, // Use fallbacks if download fails
    });
  }

  /**
   * Clean up unused assets
   */
  async pruneUnusedAssets(): Promise<void> {
    // Remove skill packs not used in 30 days
    const unusedPacks = await this.getUnusedPacks(30);

    for (const pack of unusedPacks) {
      if (pack.type === "skill") {
        await this.removePackage(pack.id);
      }
    }
  }

  /**
   * Preload assets for upcoming session
   */
  async preloadForSession(sessionPlan: SessionPlan): Promise<void> {
    const requiredPacks = sessionPlan.skills.map((s) => s.assetPack);

    // Download in background before session starts
    for (const pack of requiredPacks) {
      if (!(await this.hasPackage(pack))) {
        this.downloadPackage({
          id: pack,
          priority: "medium",
          background: true,
        });
      }
    }
  }
}
```
